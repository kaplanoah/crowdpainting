## To launch app

Online: visit [crowdsourcedpainting.com](http://crowdsourcedpainting.com/)

Locally: From terminal, in app's directory, run:

    git pull
    bundle install
    rake db:migrate
    rails s

In web browser, visit:

    http://localhost:3000/
